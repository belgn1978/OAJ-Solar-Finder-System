# Adapter Design

## Purpose

The adapter connects the solar finder to a specific telescope.

Only the adapter changes between telescope models.

The rest of the system remains identical.

---

## Supported Telescopes

- Nacatin 90/800 (Current)
- Sky-Watcher (Future)
- Celestron (Future)
- SVBONY (Future)
