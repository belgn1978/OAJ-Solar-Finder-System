# Interface Design

## Purpose

The interface is the heart of the OAJ Solar Finder System.

Every telescope adapter connects to the main body using exactly the same interface.

This means the main body never needs redesigning when supporting a new telescope.

---

## Design Goals

- Universal connection
- Easy to print
- Strong
- Minimal supports
- Beginner friendly
- Replaceable adapters

---

## Version

Current Version: 0.1

---

## Ideas

- Male dovetail on adapter
- Female socket in main body
- Positive stop
- Future locking clip

---

## Future Improvements

- Locking button
- Magnetic retention
- Quick release
